# 我的餐廳清單

![image](https://github.com/Shiang1214/restaurant_list/blob/main/截圖%202023-03-19%2022.15.43.png)


## 介紹
可以查看每間餐廳的類型與評分，選擇有興趣的餐廳確認詳細的介紹、聯絡方式、地址等等

## 使用方式
確認有安裝 Node.js 與 npm
下載檔案後在終端機上以 npm run dev 執行

## 環境建置
Node.js v14.16.0
Express v4.16.4
Express-handlebars v3.0.0
bootstrap v5.1.3
Font-awesome 5.8.1
